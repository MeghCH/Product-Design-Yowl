## /docs/06_prototype — README

This folder contains the interactive prototype of the application in desktop version and mobile version.

It includes a Figma link to a clickable prototype.

The prototype demonstrates the main user flow and key features.

**The supported interactions include:**

- Navigation between main screens (home, login, book page, book reviews).

- Viewing book details.

- Adding a review.

- Viewing the books profile list of another person.

**_This prototype is used to test usability, validate the user journey, and gather feedback before development._**
