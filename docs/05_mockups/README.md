## /docs/05_mockups 

This folder contains the high-fidelity mockups of the application.

- At least three main screens are presented with detailed visual design.
- These mockups include colors, typography, icons, spacing, and visual hierarchy.

**At the end of the documentation there is:**

- A short justification explaining key design choices (layout, colors, components, navigation).
- An explanation of accessibility decisions.

**Accessibility considerations include:**

- Sufficient color contrast for readability.

- Clear typography and readable font sizes.

- Consistent navigation and identifiable interactive elements.

**_These mockups represent the final visual direction of the product before development._**
