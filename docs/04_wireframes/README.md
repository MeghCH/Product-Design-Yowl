## /docs/04_wireframes - README

This folder contains the low-fidelity wireframes of the main screens of the application.

The wireframes represent the core user flow and main interfaces of the product.

They are created in low fidelity to focus on structure, layout, and navigation rather than visual design.

**The wireframes cover all main screens, such as:**

- Home page

- Login page

- Profile page

- Books profile page

- Book info page

- Book reviews page

**_They are used to validate the user journey, test usability, and ensure that the main features are clearly organized before moving to high-fidelity design._**
