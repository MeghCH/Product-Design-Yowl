## /docs/02_personas — User Journey Summary

This document describes the main user scenario from first visit to personalized recommendations.

The user arrives on the home page, logs in, and accesses their profile with history and saved content.
They search for a movie or select one from their history, view its details, and leave a review.
After reviewing, the system generates personalized suggestions based on their tastes and behavior.
The user can add suggested movies to a personal list and later view, sort, or remove them.

**_This journey shows how the product helps users remember what they watched and discover new content easily._**
