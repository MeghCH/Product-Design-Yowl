# /docs/09_pitch - README

A unified app to **track and share** cultural works: **Books · Video Games · Movies · TV Series**.

## Problem

Cultural tracking is fragmented across multiple platforms (IMDb, Letterboxd, Goodreads, Steam, etc.). There’s no simple way to get a **global view** of what you consume across formats.

## Concept

Users can:

- Search works (with type filters)
- Mark as **To Watch / Seen**
- Add a **rating** and a short **review**
- Browse **public profiles** and see others’ reviews

## MVP

Includes:

- Authentication
- Search + work details
- Status (To Watch / Seen)
- Ratings + short reviews
- Personal library
- Public user profiles

Out of scope:

- Follow/feed/notifications, comments
- Recommendations
- Advanced tracking (episodes/pages/playtime)
- Moderation/monetization
